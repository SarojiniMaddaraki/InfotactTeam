## 📅 January(1st Month) Project – AdVantage Gen

### 📌 Overview

**AdVantage Gen** is an automated social media campaign generation platform.

The system allows users to enter a **single prompt** and automatically generate a complete marketing campaign asset including:

- AI generated promotional image
- Social media caption
- Relevant hashtags
- Branding elements like logo and call to action buttons

The generated content is optimized for platforms such as **Instagram and LinkedIn**, helping marketers create campaigns faster.

---

### ✨ Key Features

- 🎨 AI image generation using Hugging Face models  
- ✍️ Caption and hashtag generation using Gemini  
- 🧠 Prompt enhancement for improved image quality  
- 🏷️ Automatic logo overlay and CTA button placement  
- 🖥️ Campaign editing interface  
- 💾 Campaign storage and history  
- 🔁 Remix feature for campaign variations

## 📅 February(2nd Month) Project – OPSMIND AI

### 📌 Overview

**OPSMIND AI** is an intelligent SOP assistant that allows users to interact with documents using AI.

Users can upload SOP documents and ask questions. The system performs **semantic search** and generates accurate responses based on the document content using a **Retrieval-Augmented Generation (RAG)** architecture.

---

### ✨ Key Features

- 📄 Document upload and processing  
- 🔍 Semantic search using embeddings  
- 🤖 AI chatbot with contextual responses  
- 📚 Citation based answers from documents  
- ⚡ Real time response streaming  
- 💾 Chat history persistence  
- 🔐 Secure backend architecture
